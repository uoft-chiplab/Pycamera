Class pixis_camera.PixisCamera

Class that implements a Pixis camera. The attributs of the class describe the camera and are used to create a graphical representation.

High level methods are provided to control the camera.

Methods

  • acquire(self)

    Captures the sequence. This is a blocking call, it does not return until the images are captured and transfered. This uses the attributs of the object to derive the parameters of the acquisition.

  • close(self)

    Call to close the camera.

  • close_shutter(self)

    Close the shutter right away.

  • get_buffer_size(self)

    Sets the buffer size by querying lib_pixis. It is necessary to do This before an acquisition, as python needs to provide the right amount of memory to the C routines.

  • open(self)

    Call to initialiase the camera.

  • open_shutter(self)

    Opens the shutter right away.

  • triggered_close_shutter(self)

    Close the shutter at the next trigger.

  • triggered_open_shutter(self)

    Opens the shutter at the next trigger.
